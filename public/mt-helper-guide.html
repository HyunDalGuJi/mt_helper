게스트

호스트가 등록해준 이름으로 입장해, 게임에 참여하고 정보를 확인합니다

입장하기

  • 로그인 화면에서 게스트를 선택하고, 호스트가 등록해준 이름을 입력하면 입장됩니다.
  • 비밀번호는 본인 이름과 동일합니다. 비워두면 자동으로 채워져요.
  • "등록되지 않은 참가자"라고 나오면 호스트에게 등록을 요청하세요. 띄어쓰기·오타까지 정확히 같아야 합니다.

무엇을 할 수 있나요

  • 📢 공지 확인 — 홈 상단 배너나 분위기&기타 → 공지에서 호스트 공지를 봅니다. 안 읽은 공지엔 New! 표시가 붙어요.
  • 🎲 미니게임 참여 — 호스트가 문제를 내면(업다운·초성·자유출제) 답을 제출합니다.
  • 🗳️ 투표 — 진행 중인 투표에 참여합니다. 1인 1표이고, 마감 전에는 선택을 바꿀 수 있어요.
  • 👀 결과 보기 — 룰렛 당첨, 팀 배정(내 팀 ⭐), 점수판 순위, N빵 정산(내 금액 ⭐), 일정표를 확인합니다.

알아두면 좋아요

  • 화면은 10초마다 자동 갱신됩니다. 즉시 새로고침하려면 상단 🔄 버튼을 누르세요.
  • 편집·진행은 호스트만 할 수 있어요. '👀 보기 전용' 항목은 눈으로 확인하는 용도입니다.
  • 다른 기기·브라우저로 다시 접속해도 이름만 넣으면 내 투표·제출 기록이 그대로 유지됩니다.
  • 🌙 버튼으로 다크 모드를 켤 수 있습니다 (내 기기에만 적용).
💡 로그인 화면과 앱 상단의 ❓ 버튼을 누르면 언제든 이 사용법을 다시 볼 수 있어요.
👑

호스트

MT 전반을 준비하고 진행합니다. 참가자 등록부터 게임·정산까지 대부분의 편집 권한을 가집니다

처음 시작하기

  • 로그인 화면에서 호스트를 선택하고 비밀번호를 정해 입력하세요. 최초 입력한 비밀번호가 그대로 설정됩니다 (4자 이상).
  • 이후에는 그 비밀번호로 로그인합니다. 잊어버리면 Admin이 초기화해줄 수 있어요.
  • 가장 먼저 ⚙️ 관리 → 참가자 계정에서 이름을 등록하세요. 쉼표로 여러 명을 한 번에 넣을 수 있습니다.

게임 & 진행

  • 🎯 룰렛 — 등록된 참가자 중 랜덤 1명을 뽑습니다. 결과가 게스트에게도 공유됩니다.
  • 👥 팀배정 — 팀 수를 정하고 자동으로 섞어 배정합니다.
  • 🏆 점수판 — 팀을 만들고 라운드별 점수를 입력하면 순위가 자동 집계됩니다.
  • 🎲 미니게임 — 업다운·초성은 문제를 내면 게스트가 답을 제출하고, 자유출제는 직접 문제를 쓰고 정답자를 선택합니다. 정답은 호스트에게만 보여요.

정산 & 관리

  • 🛒 장바구니 — 마트·온라인·기증·사전구매로 나눠 관리합니다. 엑셀 내보내기/가져오기가 가능해요.
  • 🍶 주류계산 — 인원과 1인당 소주량으로 필요한 술을 계산합니다.
  • 💸 N빵 — 지출 항목별 금액과 제외 인원을 정하면 1인당 정산액이 나옵니다.
  • 📅 일정 — 시간과 내용을 입력하면 게스트가 볼 수 있는 타임테이블이 됩니다.

분위기 & 공지

  • 📢 공지 — 텍스트와 이미지로 공지를 올립니다. 이미지는 자동으로 축소 저장돼요.
  • ❓ 아이스브레이킹 — 질문을 뽑습니다. 한 번 나온 질문은 다시 안 나오며, 초기화하면 전부 다시 나옵니다.
  • 🗳️ 투표 — 주제와 선택지로 투표를 만들고 마감/재개할 수 있습니다.

진행 팁

  • 호스트가 게임에 직접 참여하려면, 참가자 계정에 본인 이름을 등록한 뒤 시크릿 창에서 게스트로도 로그인하세요.
  • 게임 데이터를 정리하려면 ⚙️ 관리 → 참가자 계정 하단의 '게임 데이터 초기화'를 쓰세요. 참가자 계정과 비밀번호는 유지됩니다.
⚠️ 여러 사람이 동시에 호스트로 편집하면 마지막 저장이 우선됩니다. 편집은 한 사람이 맡는 게 안전해요.
⚙️

Admin

시스템 관리자. 호스트의 모든 권한에 더해 백업·복원·시스템 초기화를 담당합니다

입장하기

  • 로그인 화면에서 Admin을 선택하고 관리자 비밀번호를 입력합니다.
  • 이 비밀번호는 서버 설정(ADMIN_PASSWORD 환경변수)으로 관리됩니다. 앱 데이터가 아니라 Cloudflare 설정이에요.
  • Admin으로 관리 카테고리에 들어가면 🖥️ 시스템 탭이 먼저 열립니다.

시스템 (Admin 전용)

  • 🖥️ System Info — 저장 용량, 참가자·공지·투표 수, 호스트 비번/테마 설정 여부를 확인합니다.
  • 💾 백업(Export) — 전체 데이터를 JSON 파일로 내려받습니다. MT가 끝난 뒤 아카이브로 보관할 수 있어요.
  • 📥 복원(Import) — 백업 JSON을 올려 그 시점 상태로 되돌립니다.
  • 🔑 호스트 비번 초기화 — 호스트가 비번을 잊었을 때 초기화합니다. 다음 호스트 로그인 시 새 비번이 설정돼요.
  • 💥 시스템 전체 초기화 — 참가자 계정까지 모두 삭제합니다. 새 MT를 시작할 때 사용하세요.

테마 설정

  • 🎨 앱 테마 — 로그인 화면의 배경 이미지, 팀명, MT 기간, 장소를 설정합니다. 배경 이미지는 자동 축소됩니다.
  • 설정 후 로그아웃하면 로그인 화면에서 "○○ MT에 오신 것을 환영합니다!" 문구와 함께 확인할 수 있어요.

참가자 계정 관리

  • Admin이 참가자 계정 탭을 열려면 호스트 비밀번호가 필요합니다 (개인정보 보호). 브라우저를 닫으면 다시 잠겨요.
  • 이 잠금은 관리 화면에만 적용됩니다. 팀배정·정산 등 다른 화면과 백업 파일에는 이름이 포함됩니다.

게임 참여 팁

  • Admin도 게임에 끼려면 참가자 계정에 이름을 등록하고, 시크릿 창에서 게스트로 로그인하면 됩니다.
  • 세 역할은 로그인 단위로 구분되며, 시크릿 창이나 다른 브라우저로 여러 역할을 동시에 쓸 수 있습니다.

자주 묻는 질문

화면이 안 바뀌어요 / 최신 내용이 안 보여요

화면은 10초마다 자동 갱신됩니다. 더 빨리 보려면 앱 상단의 🔄 버튼을 누르세요. 게스트에게는 호스트가 방금 한 작업이 몇 초 뒤에 반영됩니다.

게스트 로그인이 "등록되지 않은 참가자"라고 나와요

호스트가 참가자 계정에 그 이름을 먼저 등록해야 합니다. 이미 등록했다면 띄어쓰기나 오타가 없는지 확인하세요. 이름과 완전히 똑같이 입력해야 합니다.

호스트 비밀번호를 잊어버렸어요

Admin이 시스템 탭에서 '호스트 비밀번호 초기화'를 실행하면, 다음 호스트 로그인 때 새 비밀번호를 다시 정할 수 있습니다.

한 사람이 호스트도 하고 게임에도 참여할 수 있나요

가능합니다. 참가자 계정에 본인 이름을 등록한 뒤, 일반 창은 호스트로, 시크릿 창(또는 다른 브라우저)은 게스트로 로그인하면 됩니다. 로그인은 브라우저 단위로 구분됩니다.

지난 MT 기록을 보관하고 싶어요

Admin이 시스템 탭에서 'DB Export'로 전체 데이터를 JSON 파일로 내려받아 보관하면 됩니다. 나중에 'Import'로 그 파일을 올리면 당시 상태가 그대로 복원됩니다.

다크 모드는 모두에게 적용되나요

아니요. 다크 모드는 각자 기기에만 적용되는 개인 설정입니다. 🌙 버튼으로 언제든 켜고 끌 수 있어요.
